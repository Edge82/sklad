# 📦 Инструкция по установке и запуску приложения на Windows

## Требования
- Windows 10 или выше
- Node.js 18+ (скачай с https://nodejs.org/)
- PowerShell или командная строка

## 🚀 Быстрый старт (2 шага)

### 1. Установи Node.js
- Перейди на https://nodejs.org/
- Скачай **LTS версию** (18.x или 20.x)
- Запусти установщик и следуй инструкциям
- При установке отметь "Add to PATH"

### 2. Запусти приложение

1. Распаковай папку `sklad` на диск (например `C:\Users\ТвоеИмя\sklad`)
2. **Двойной клик на `run.bat`** — и всё!

Скрипт автоматически:
- ✅ Создаст файл конфигурации `.env` (если его нет)
- ✅ Установит зависимости
- ✅ Соберет приложение
- ✅ Запустит сервер

## ⚙️ Настройка подключения к 1С

**ВАЖНО!** После первого запуска проверь параметры подключения к 1С:

1. Открой файл `.env` в блокноте
2. Найди строки:
```
ONEC_BASE_URL=https://msk1.1cfresh.com/a/sbm/3784912/odata/standard.odata
ONEC_LOGIN=odata.user
ONEC_PASSWORD=HoroshkoUserSklad
WAREHOUSE_GUID=344cfb30-e233-11f0-862e-fa163e5c9fa8
```

3. Если это не твои данные - **замени на свои реальные значения**
4. Сохрани файл
5. Перезагрузи приложение (останови и заново запусти `run.bat`)

## 🔧 Устранение проблем

### Ошибка: "node не является внутренней или внешней командой"
- Переустанови Node.js
- Убедись что при установке отметил "Add to PATH"
- Перезагрузи PowerShell/CMD

### Ошибка: "Port 8000 already in use"
Отредактируй файл `.env`:
```
BACKEND_PORT=3000
```
И перезагрузи приложение.

### Ошибка подключения к 1С
1. Проверь правильность параметров в `.env`
2. Убедись что 1С доступна по сети
3. Проверь логин и пароль
4. Смотри логи в окне приложения

## 💾 Резервная копия данных

База данных хранится в папке `/data/app.db`

**Регулярно создавай резервные копии:**
- Скопируй папку `/data` в безопасное место
- Перед обновлением приложения сохрани `/data/app.db`

Для восстановления - просто скопируй старый `app.db` обратно в папку `/data`
