# 📄 Генерация PDF-версии руководства

## Способ 1: Автоматическая генерация (рекомендуется)

### Требования
- Node.js 16+ (проверка: `node --version`)
- npm (идёт в комплекте с Node.js)

### Инструкция

**Шаг 1. Установка зависимостей**

Откройте терминал в папке `docs/pdf-generator` и выполните:

```bash
npm install
```

**Шаг 2. Генерация PDF**

```bash
npm run generate
```

**Результат:**
- Файл `docs/USER_MANUAL.pdf` — готовое руководство
- Время генерации: ~5-10 секунд
- Размер файла: ~2-3 МБ (со скриншотами)

---

## Способ 2: Расширение VS Code (альтернатива)

Если не хотите использовать Node.js:

### Установка расширения

1. Откройте VS Code
2. Перейдите в **Extensions** (`Ctrl+Shift+X`)
3. Найдите и установите: **`Markdown PDF`** (автор: `yzane`)

### Генерация PDF

1. Откройте файл `USER_MANUAL.md` в VS Code
2. Нажмите `Ctrl+Shift+P` (командная палитра)
3. Введите: `Markdown PDF: Export (pdf)`
4. Нажмите `Enter`

**Результат:**
- Файл `USER_MANUAL.pdf` появится рядом с `USER_MANUAL.md`

## Распространённые проблемы

### ❌ Ошибка: `Cannot find module 'puppeteer'`

**Решение:**
```bash
npm install
```

### ❌ Ошибка: `node --version` показывает версию < 16

**Решение:**
- Скачайте актуальную Node.js с [nodejs.org](https://nodejs.org/)
- Установите LTS версию

### ❌ Скриншоты не отображаются в PDF

**Проверьте:**
1. Скриншоты лежат в папке `docs/screenshots/`
2. Все файлы изображений существуют

### ❌ PDF слишком большой (>10 MB)

**Решение:**
- Сожмите скриншоты перед генерацией
- Используйте инструмент [TinyPNG](https://tinypng.com/) для PNG

## Для разработчиков

### Структура файлов

```
docs/
├── pdf-generator/
│   ├── generate-pdf.js      # Скрипт генерации
│   ├── package.json         # Зависимости Node.js
│   ├── styles.css           # Стили для PDF (встроены в JS)
│   └── README.md            # Эта инструкция
├── screenshots/             # Изображения для руководства
│   ├── main-page.png
│   ├── inventory-page.png
│   └── ...
└── USER_MANUAL.pdf          # Сгенерированный PDF (результат)
```

### Изменение стилей

Откройте `generate-pdf.js` и отредактируйте CSS в переменной `fullHtml`:
- Шрифты (`font-family`)
- Цвета (`color`, `background-color`)
- Размеры полей (`margin`, `padding`)

---

*Инструкция актуальна для версии от 2025 года.*
