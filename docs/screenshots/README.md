## 📐 Требования к скриншотам

### Разрешение и формат
- **Формат**: PNG (рекомендуется) или JPG
- **Разрешение**: 1920×1080 или выше
- **Масштаб**: 100% (не масштабировать)
- **Качество**: Высокое, текст должен быть читаемым

### Что должно быть видно
1. **Полная страница** — захватывайте всю видимую область
2. **Все элементы управления** — кнопки, фильтры, поля поиска
3. **Данные** — заполните таблицу тестовыми данными (5-10 строк)
4. **Статусы** — покажите разные статусы (цвета)

### Не рекомендуется
- ❌ Не обрезайте важные элементы
- ❌ Не размывайте конфиденциальные данные (лучше замените на тестовые)
- ❌ Не используйте скриншоты с телефона
- ❌ Не делайте скриншоты с открытыми личными данными

## 🛠️ Инструменты для создания скриншотов

### Windows
| Инструмент | Горячие клавиши | Описание |
|------------|-----------------|----------|
| **Ножницы** | `Win + Shift + S` | Стандартное приложение |
| **Snip & Sketch** | `Win + Shift + S` | Расширенный инструмент |
| **Lightshot** | `Print Screen` | Стороннее приложение |
| **ShareX** | Настраиваемые | Профессиональный инструмент |

### macOS
| Инструмент | Горячие клавиши | Описание |
|------------|-----------------|----------|
| **Весь экран** | `Cmd + Shift + 3` | Скриншот всего экрана |
| **Область** | `Cmd + Shift + 4` | Выделение области |
| **Окно** | `Cmd + Shift + 4, затем пробел` | Скриншот окна |

### Linux
| Инструмент | Горячие клавиши | Описание |
|------------|-----------------|----------|
| **GNOME Screenshot** | `Print Screen` | Стандартное приложение |
| **Flameshot** | Настраиваемые | Продвинутый инструмент с аннотациями |
| **Shutter** | Настраиваемые | Профессиональный инструмент |

## ✅ Чек-лист перед сохранением

- [ ] Все текстовые поля заполнены осмысленными данными
- [ ] Нет личных данных (ФИО, телефоны, email)
- [ ] Все кнопки и элементы видны
- [ ] Таблицы содержат 5-10 строк для наглядности
- [ ] Статусы показаны разными цветами
- [ ] Разрешение не менее 1920×1080
- [ ] Файл назван согласно таблице выше
- [ ] Файл сохранён в `docs/screenshots/`

## 💡 Советы

1. **Используйте режим инкогнито** для чистого интерфейса без расширений
2. **Скройте браузерные панели** (`F11` — полный экран)
3. **Установите масштаб 100%** в браузере (`Ctrl + 0`)
4. **Заполните данные заранее** — создайте тестовые материалы, заказы, сотрудников
5. **Покажите разные состояния** — например, заказы с разными статусами
