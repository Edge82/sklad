## 🔍 Поиск информации

### В PDF (Adobe Reader, браузер):
1. Нажмите `Ctrl+F`
2. Введите ключевое слово (например, "QR", "сканирование", "печать")

### По номеру рисунка:
- В оглавлении PDF найдите "Рисунок X"
- Перейдите на указанную страницу

---

## 🖨️ Печать руководства

Если нужна бумажная версия:

1. Откройте `docs/USER_MANUAL.pdf`
2. Нажмите `Ctrl+P` (Печать)
3. Выберите настройки:
   - **Двусторонняя печать** — экономия бумаги
   - **Цветная** — для скриншотов
   - **A4** — стандартный формат

## 📱 Просмотр на устройстве

### На компьютере:
- Откройте PDF в любом браузере или Adobe Reader

### На планшете/телефоне:
- Скопируйте файл `docs/USER_MANUAL.pdf` на устройство
- Откройте в приложении для PDF

---

## 🔄 Обновление руководства

Если изменился интерфейс или добавились функции:

1. Обновите скриншоты в папке `docs/screenshots/`
2. Отредактируйте `USER_MANUAL.md`
3. Перегенерируйте PDF:
   ```bash
   cd docs/pdf-generator
   npm run generate
   ```
