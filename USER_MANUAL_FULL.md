# 📦 Полное Руководство Пользователя Складской Системы

## 🎯 О Системе

Система управления складом и производством для мебельного предприятия. Позволяет:
- ✅ Управлять складскими запасами (материалы и готовые изделия)
- ✅ Работать с заказами клиентов и перемещениями
- ✅ Отслеживать производство через QR-коды
- ✅ Контролировать использование материалов и инструментов
- ✅ Формировать отчёты по деятельности

---

## 🚀 Быстрый Старт

### 1. Вход в Систему

1. Откройте браузер и перейдите на адрес склада (например: `http://localhost:3000`)
2. Введите логин и пароль, выданные администратором
3. Нажмите **Войти**

**Тестовые учётные данные (для разработки):**
| Роль | Логин | Пароль |
|------|-------|--------|
| Директор | `admin` | `admin` |
| Менеджер | `manager` | `manager` |
| Кладовщик | `storekeeper` | `storekeeper` |
| Рабочий | `worker` | `worker` |

### 2. Первый Заход

После первого входа:
1. Проверьте профиль (справа вверху → **Личный кабинет**)
2. Измените пароль при необходимости
3. Ознакомьтесь с интерфейсом

---

## 👥 Роли и Права Доступа

### Директор / Менеджер (Полный доступ)
- ✅ Видят все цены и стоимость
- ✅ Управляют сотрудниками (создание, редактирование, удаление)
- ✅ Доступны вкладки **Сотрудники** и **Отчеты**
- ✅ Могут менять пароли другим сотрудникам
- ✅ Полный доступ ко всем функциям

### Кладовщик / Рабочий (Ограниченный доступ)
- ❌ **Не видят цены** (вместо цен — дефис `-`)
- ❌ **Не видят** вкладки **Сотрудники** и **Отчеты**
- ❌ **Не могут** создавать/редактировать сотрудников
- ✅ Могут работать со складом и заказами
- ✅ Могут сканировать QR-коды
- ✅ Могут менять **свой** пароль в профиле

### 1. 🏠 Склад ТМЦ

**Где найти:** Главное меню → **Склад** → **Склад ТМЦ**

#### Что здесь можно сделать:

| Действие | Как выполнить | Для кого доступно |
|----------|---------------|-------------------|
| Просмотр материалов | Открываете страницу | Все |
| Поиск по материалам | Поле поиска сверху | Все |
| Фильтрация | Выбрать категорию/склад/статус | Все |
| Добавить материал | Кнопка **"Новый материал"** | Только директор/менеджер |
| Редактировать материал | Клик на строку материала | Только директор/менеджер |
| Переместить материал | Кнопка **"Перемещение"** | Все |
| Посмотреть историю | В карточке материала → вкладка **"История"** | Все (без цен) |
| Печать QR-кода | Кнопка **"QR"** в карточке | Все |
| Экспорт в Excel | Кнопка **Excel** | Все |
| Экспорт в PDF | Кнопка **PDF** | Все |

#### Статистика на странице:

- **Всего позиций** — количество SKU (артикулов)
- **Отсутствует** — материалы с нулевым остатком
- **Зарезервировано** — количество зарезервированных единиц
- **В наличии** — материалы со складским остатком
- **Стоимость запасов** — общая стоимость (только для director/manager)

#### Фильтры:

- **Категория** — древесина, фурнитура, крепёж и т.д.
- **Склад** — выбор конкретного склада
- **Статус** — в наличии, отсутствует, мало осталось, в пути и др.
- **Поиск** — по названию или артикулу

### 3. 📝 Заказы

**Где найти:** Главное меню → **Заказы**

#### Что здесь можно сделать:

| Действие | Как выполнить | Для кого доступно |
|----------|---------------|-------------------|
| Просмотр заказов | Открываете страницу | Все |
| Поиск по заказу | По номеру или клиенту | Все |
| Фильтрация по статусу | Выбрать статус | Все |
| Открыть заказ | Клик на строку заказа | Все |
| Синхронизация с 1С | Кнопка **"Синхронизация с 1С"** | Все |
| Сгенерировать QR-коды | В заказе → **"Генерировать QR"** | Все |
| Просмотреть детали | Кнопка **"Глаз"** в заказе | Все |
| Изменить окраску | В заказе → блок **"Окраска"** | Все |

#### Статусы заказов:

- **В работе** (`in_progress`) — заказ в производстве
- **На складе** (`ready`) — готов к отгрузке
- **Отгружен** (`shipped`) — передан клиенту

#### Работа с Заказом:

**Шаг 1: Откройте заказ**
- Кликните на строку заказа
- Откроется детальное представление

**Шаг 2: Проверьте позиции**
- На вкладке **"Позиции заказа"** видите список товаров
- Для каждого товара показан прогресс: `отсканировано / всего QR-кодов`

**Шаг 3: Сгенерируйте QR-коды (если нужно)**
1. Нажмите кнопку **"Генерировать QR коды"**
2. Укажите:
   - **Товар** — выберите из списка позиций заказа
   - **Количество** — сколько кодов создать
   - **Сотрудник** — кто выполняет операцию
3. Нажмите **OK** — коды созданы

**Шаг 4: Измените окраску (если нужно)**
1. В правой панели найдите блок **"Окраска"**
2. Нажмите **"Редактировать"**
3. Введите значение (например: *"Красная матовая"*)
4. Нажмите **"Сохранить"**
5. Изменение логируется и синхронизируется с 1С

#### Статистика на странице:

- **Всего заказов** — количество заказов
- **В работе** — заказы в производстве
- **На складе** — готовые заказы
- **Отгружен** — переданные клиенту
- **Стоимость заказов** — сумма заказов в работе (только для director/manager)

**Где найти:** Главное меню → **Сотрудники**

#### Что здесь можно сделать:

| Действие | Как выполнить | Для кого доступно |
|----------|---------------|-------------------|
| Просмотр списка | Открываете страницу | Только директор/менеджер |
| Поиск по сотруднику | Поле поиска | Только директор/менеджер |
| Фильтрация по отделу | Выбрать отдел | Только директор/менеджер |
| Добавить сотрудника | Кнопка **"Добавить сотрудника"** | Только директор/менеджер |
| Редактировать | Клик на сотрудника → **"Изменить"** | Только директор/менеджер |
| Удалить | Кнопка **"Удалить"** | Только директор/менеджер |
| Сменить пароль | Кнопка **"Ключ"** | Только директор/менеджер |
| Просмотреть историю | В карточке → вкладка **"История"** | Только директор/менеджер |

#### Статистика на странице:

- **Всего сотрудников** — общее количество
- **Активных** — работают сейчас
- **В отпуске** — находятся в отпуске
- **На больничном** — на больничном
- **ФОТ в месяц** — фонд оплаты труда (только для director/manager)

#### Работа с Сотрудником:

**Шаг 1: Создайте сотрудника**
1. Нажмите **"Добавить сотрудника"**
2. Заполните форму:
   - **ФИО** — обязательное поле
   - **Email** — для уведомлений
   - **Телефон** — контактный номер
   - **Должность** — например, *"Мастер цеха"*
   - **Отдел** — выберите из списка
   - **Роль** — admin/manager/worker/warehouse/production
   - **Статус** — active/inactive/vacation/sick
   - **Зарплата** — размер оклада
3. Нажмите **"Сохранить"**
4. Система создаст учётную запись с временным паролем `12345678`
5. Покажите сотруднику логин и пароль для первого входа

**Шаг 2: Смените пароль сотруднику**
1. Найдите сотрудника в списке
2. Нажмите кнопку **"Ключ"** (смена учётных данных)
3. Введите новый **логин** и **пароль**
4. Нажмите **"Сохранить"**
5. Сообщите сотруднику новые данные

**Шаг 3: Просмотрите историю операций**
1. Кликните на сотрудника
2. Откроется карточка с деталями
3. Перейдите на вкладку **"История операций"**
4. Видите последние 10 действий:
   - Генерация QR-кодов
   - Сканирование
   - Изменение окраски
   - Удаление кодов

### 8. 📊 Отчёты

**Где найти:** Главное меню → **Отчеты**

**⚠️ Доступно только для директора и менеджера!**

#### Разделы отчётов:

**1. Топ сотрудников (активность)**
- Показывает 5 самых активных сотрудников
- Количество операций за период
- Можно перейти в профиль сотрудника

**2. Заказы и материалы**
- Все заказы с составом материалов
- Кто использовал материалы (заказчики)
- Общая стоимость заказов

**3. Списание на изделия**
- Какие материалы пошли на какие изделия
- Количество списанных единиц
- Динамика использования

**4. Движение материалов**
- История прихода/расхода материалов
- Кто и когда использовал
- Суммы операций

#### Фильтры в отчётах:

- **По сотруднику** — выбрать конкретного
- **По инструменту** — фильтр по инструменту
- **По дате** — выбрать период
- **По типу операции** — приход/расход

## 📊 Глоссарий

| Термин | Значение |
|--------|----------|
| **QR-код** | Уникальный код для отслеживания товара в производстве |

| **Номенклатура** | Каталог материалов и товаров в 1С |

## ⚠️ Частые Вопросы

### Общая информация

**Q: Почему не видны цены?**  
**A:** У вашей роли нет доступа к ценам. Обратитесь к директору или менеджеру.

**Q: Почему нет вкладки "Сотрудники"?**  
**A:** Эта вкладка доступна только директору и менеджеру.

**Q: Как изменить свой пароль?**  
**A:** Перейдите в **Личный кабинет** → кнопка **"Сменить пароль"** → введите текущий и новый пароль.

---

### Работа с заказами

**Q: Почему не видны QR-коды после перезагрузки?**  
**A:** QR-коды хранятся на сервере. Если пропали:
1. Проверьте, что в том же заказе
2. Обновите страницу (F5)
3. Если не помогло — обратитесь к администратору

**Q: Как исправить ошибочно отсканированный код?**  
**A:** Обратитесь к администратору — он может изменить статус кода.

**Q: Где посмотреть историю изменения окраски?**  
**A:** Откройте заказ → проверьте историю операций. Все изменения логируются.

### Работа со складом

**Q: Как добавить новый материал?**  
**A:** Только директор/менеджер: **Склад ТМЦ** → кнопка **"Новый материал"** → заполните форму.

**Q: Как посмотреть, где хранится материал?**  
**A:** Откройте карточку материала → поле **"Место хранения"**.

**Q: Как экспортировать данные склада?**  
**A:** Используйте кнопки **Excel** или **PDF** на странице склада.

### Работа с сотрудниками

**Q: Как создать нового сотрудника?**  
**A:** Только директор/менеджер: **Сотрудники** → **"Добавить сотрудника"** → заполните форму.

**Q: Как сменить пароль сотруднику?**  
**A:** Только директор/менеджер: найдите сотрудника → кнопка **"Ключ"** → введите новые учётные данные.

**Q: Почему не могу удалить сотрудника?**  
**A:** Удаление доступно только директору и менеджеру.

### Технические вопросы

**Q: Система не загружается**  
**A:** 
1. Проверьте подключение к сети
2. Обновите страницу (Ctrl+F5)
3. Проверьте, запущен ли backend (порт 8000)

**Q: Ошибка синхронизации с 1С**  
**A:** 
1. Проверьте настройки в **Интеграция 1С**
2. Убедитесь, что 1С доступна по сети
3. Проверьте логи в `.data/sync.log`

**Q: Данные не сохраняются**  
**A:** 
1. Проверьте подключение к сети
2. Откройте консоль браузера (F12) и посмотрите ошибки
3. Обратитесь к администратору
